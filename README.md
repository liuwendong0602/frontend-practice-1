# 前端练习项目

这是一个简单的前端练习项目，展示了基础的 HTML、CSS 和 GitHub Pages 使用方法。

## 使用的技术：
- HTML
- CSS
- GitHub Pages

## 项目描述：
该项目展示了一个简单的布局，包括导航栏、内容区域和页脚。它使用了现代 CSS 技术，如 Flexbox 和 CSS Grid，设计为响应式网页。

## 在线演示：
你可以通过以下链接查看该项目的在线演示：  
[在线演示](https://liuwendong0602.github.io/frontend-practice-1/)

## 获取项目：
1. 克隆该仓库：
    ```bash
    git clone https://github.com/liuwendong0602/frontend-practice-1.git
    ```
2. 在浏览器中打开 `index.html` 文件。
